# RNV2ex
[![996.icu](https://img.shields.io/badge/link-996.icu-red.svg)](https://996.icu)

基于 React-Native & Redux 的 v2ex 客户端


☑️ 项目是16年写的，近期打算重构一下

#### Usage

- git clone
- npm install
- react-native run-ios || react-native run-android
- npm start
